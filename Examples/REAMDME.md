This repository contains examples for running ORB-SLAM3 with the most common SLAM datasets using the new calibration format files.

For the Ubuntu 22.04 EuRoC `MH_01_easy` and TUM-VI `dataset-room1_512_16` monocular / monocular-inertial commands validated in this checkout, see `../docs/Ubuntu22_EuRoC_TUMVI.md`.
